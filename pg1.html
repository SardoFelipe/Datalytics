O que é Big Data?



  O termo Big Data Analytics, ou apenas Big Data, refere-se às enormes quantias de informação disponíveis para um negócio considerado moderno. Em grande parte de casos se trata de dados digitais – Como por exemplo a frequência que seu aplicativo é aberto ou como os clientes interagem com o seu site. – e são vindas de fontes externas ou internas à organização.


  Para conseguir estes dados normalmente são utilizados clientes, fornecedores, redes sociais ou sensores IoT. Esses dados externos são divididos conforme suas naturezas. Podendo ser dados estruturados, como listas e tabelas, ou dados não estruturados, como resultados de busca.



  Todos os tipos de dados devem ser postos em um formato estruturado para que sejam analisados em uma data warehouse ou em tecnologias como bancos de dados Hadoop e NoSQL que suportam o processamento de grandes conjuntos deles, agrupados em sistemas variados. Através desses dados, as empresas têm a oportunidade de identificar certos padrões e comportamentos considerados comuns a partir deles. Podendo, por sua vez, serem usados para informar e basear as decisões de negócios.



 Os principais aspectos presentes na Big Data são chamados 5 V’s, sendo estes:

Volume

Variedade

Valor

Veracidade

Velocidade

Os aspectos de Volume, Variedade e Velocidade, dizem respeito à grande quantidade de dados não estruturados que devem ser analisados pelas soluções de Big Data a uma grande velocidade. O V de Veracidade é sobre as fontes e qualidade dos dados, afinal, os mesmos devem ser confiáveis. Quanto ao V de Valor, é relacionado aos benefícios que as soluções da Big Data trarão para a empresa. Cada instituição necessita checar os benefícios para ver se compensam os altos investimentos nas soluções específicas da Big Data para isso.
